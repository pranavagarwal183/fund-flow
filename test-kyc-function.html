KYC Verification Test - Hybrid System

Format: ABCDE1234F (5 letters + 4 digits + 1 letter)

How the Hybrid System Works:

  1. Database Check: First searches your local database for existing KYC records
  2. External Verification: If not found locally, checks external KYC providers (CAMS KRA, NSDL, etc.)
  3. Storage: Stores external verification results for future use
  4. Response: Returns KYC status with source information

Response Sources:

  • Database - Found in local database
  • External - Verified from external provider
  • New Record - Stored after external verification
